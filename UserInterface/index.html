# 用户界面

Wrap3 用户界面由四个主要部分组成：

  • 节点图编辑面板
  • 节点参数面板
  • Viewport选项卡：Viewport3Dopen in new window、Viewport2D、Visual editor 和 Gallery
  • 时间线open in new window

# 节点图编辑器面板

节点图编辑器允许您创建、连接和更改节点参数。

提示

点击这里open in new window了解有关节点图编辑器的更多信息。

# 节点参数面板

每当您选择一个节点时，您都可以在此面板中浏览该节点的参数。更改任何参数都会导致节点重新计算。

# Viewport选项卡

每个节点根据其类型，可以在多个地方展示，例如3D-viewportopen in new window或2D-viewport。例如，加载几何体（LoadGeom）open in new window节点会被渲染到3D-viewportopen in new window中，而转换纹理（TransferTexture）open in new window节点在2D-viewport中。

如果一个节点需要用户交互，比如选择点（SelectPoints）open in new window，它也可以显示在编辑器选项卡中。

提示

使用Q、W、E、R键可以在选项卡之间快速切换。

选择一个节点后，展示该节点的所有选项卡都以蓝线突出显示。

# 时间线（Timeline）

时间线（Timeline）使我们可以批量处理扫描。加载几何体（LoadGeom）open in new window或加载图片（LoadImage）open in new window等一些节点允许加载多个对象。当切换到下一帧时，节点会从列表中加载下一个对象。
