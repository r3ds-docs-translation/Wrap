# 节点图编辑器

在Wrap中的所有操作都通过节点来实现：创建节点、连接节点、更改节点参数等。

# 节点图编辑器导航

操作行为
拖拽平移节点
点击选择节点
shift+点击把节点添加到选择
点击空白处拖拽框选选择多个节点
鼠标滚轮放大/缩小

# 创建节点

在节点图编辑器按Tab键或鼠标右键，会展示节点列表。可以点击选择要创建的节点类型。

节点列表菜单包裹所有可创建的节点类型。节点类型包括：几何体（Geometry）、图像（Image）、选择（Selection）、对齐（Alignment）

如果输入字符，会展示一个相关节点的搜索列表。按Enter键选择节点。

# 连接节点

在第一个节点的输入或输出上按住鼠标左键，将光标拖动到第二个节点的输入或输出上，就连接了两个节点。

另一种方法是单击第一个节点的输入或输出，然后再单击第二个节点的输入或输出。

每个节点可以有多个输入，和一个输出。每个输入的颜色都与特定的类型（几何体（Geometry）、图像（Image）、多边形选择（PolygonSelection））相关，例如几何体（Geometry）是蓝色，图像（Image）是绿色。

提示

将光标在输入上悬停，可以查看他的名称和类型。

# 断开节点

右键单击连接曲线，从弹出菜单中单击断开连接（Disconnect）。

# 检查节点计算状态

每次改变节点的输入或参数，都会重新计算节点。

如果计算失败，可以在节点右侧看到一个警告标志。点击警告标志或把鼠标悬停在上面，可以看到错误信息。

# 检查节点信息

右键点击节点，在上下文菜单中可以看到**节点信息（Node Information）**操作。

点击它会展示一个节点信息对话框，包含节点的一些有用信息，如：网格中的一些顶点或多边形组，显示几何体是否为流形（manifold）的标志等。

# 重命名节点

每个节点都有个唯一名称。选择节点，按F2键可以重命名。或在节点上点击鼠标右键，在上下文菜单中选择重命名选项。

# 更改3D节点可见性

在3D-viewport中渲染的节点，如加载几何体（LoadGeom）open in new window，有个灯泡标志。单击灯泡标志切换节点在3D-viewport内的可见性。

提示

CTRL+单击灯泡标志，用来切换3D-viewport中的专有可见性。该节点会变为可见，而节点图中的其他节点将隐藏。
