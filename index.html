# 欢迎使用 Wrap 文档

Wrap 改变了处理3D扫描数据的方式。在处理大量类似对象（例如人体扫描）时，Wrap可以采用现有的basemesh并将其非刚性地拟合到每次扫描中。

Wrap 还提供了一组非常有用的扫描处理工具，例如抽取、网格过滤、纹理投影等等。借由节点图架构，只要处理了一次扫描，您就可以将此节点图应用于无限数量的其他扫描。

# Wrap4D

Wrap4D是Wrap的扩展版本，专为处理4D序列而设计。您可以在此页面上找到有关4D处理管线的更多信息。
