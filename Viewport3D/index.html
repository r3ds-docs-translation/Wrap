# Viewport3D

Viewport3D是大多数Wrap节点的渲染视图。单击对象，会选择相应的节点。视口操作与Autodesk Maya非常相似。

视口支持透视和正交投影。

提示

要更改投影类型，请到编辑（Edit）>>首选项（Preference）>>投影（Projection）

# Viewport导航

操作行为
Alt+鼠标左键旋转
Alt+鼠标中键平移
Alt+鼠标右键缩放
鼠标滚轮缩放
F键使选定对象适合视口
S键将相机的rotation与最近的世界轴对齐

# 导航类型

你可以从两种类型里选择导航方式：

  • 传统的3轴导航
  • 轨迹球（trackball）导航

轨迹球导航非常适合处理随机方向的扫描。

提示

要更改导航类型，请到编辑（Edit）>>首选项（Preference）>>导航（Navigation）
