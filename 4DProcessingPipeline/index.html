# 4D处理管线

4D表演捕捉是演员每毫米皮肤在每个时刻移动的重要数据来源。

使用深度相机、红外或彩色相机等不同的捕获方法可以实现这一点。有的使用单个摄像机（单眼捕获），有的使用多个同步摄像机从不同视角捕获演员的图像。

本文档仅介绍使用多个同步摄像机，为每个摄像机生成一组图像序列，从而捕获演员表现的视频测量方法。然后通过传统的摄影测量重建软件，基于每帧图像重建演员脸部的3D扫描。3D扫描的结果序列包含许多有用的数据，但由于每次扫描都有独特的拓扑结构和UV，因此无法直接用于生产。

下一步应该是将3D扫描序列转换为可用于生产的一致拓扑。4D处理管线的用武之地。该管线是围绕4个独立的应用程序构建的：

  • R3DS Wrap4D
  • R3DS Track
  • R3DS Rush
  • R3DS Node

# R3DS Wrap4D

Wrap4D是Wrap的扩展版本。它包含10个专门为处理4D序列而设计的特殊节点。Wrap4D将一系列带纹理的3D扫描作为输入，生成一系列具有一致拓扑结构的网格作为输出。重要的是每一帧都是独立计算的，这意味着我们可以并行处理它们。为了获得良好的效果，Wrap4D需要在演员的脸上绘制一组稀疏的标记（黑点）。标记的位置用作包裹流程的初始化，并且在处理面部的极端表情时大大提高了鲁棒性。Wrap4D也用于R3DS Track中有关眼睑和嘴唇轮廓信息的检测。

Wrap4D包括了以下用于4D处理的节点：

  • 混合稳定（BlendStabilization）open in new window
  • 编辑面部检测（EditFacialDetection）open in new window
  • 面部标注（FacialAnnotation）open in new window
  • 面部检测（FacialDetection）open in new window
  • 面部包裹（FacialWrapping）open in new window
  • 加载面部标注（LoadFacialAnnotation）open in new window
  • 加载面部检测（LoadFacialDetection）open in new window
  • 加载屏幕点（LoadScreenPoints）open in new window
  • 规划屏幕点（ProjectScreenPoints）open in new window
  • 保存面部检测（SaveFacialDetection ）open in new window

# R3DS Track

R3DS Track是一款2D追踪软件，它可以实现两个目的：

  • 追踪演员脸上的标记
  • 使用个性化面部检测器检测嘴唇和眼睑轮廓

# R3DS Node

R3DS Node是R3DS Wrap4D的无头版本，带有命令行界面。它可以与R3DS Rush结合使用，通过网络运行计算。

# R3DS Rush

R3DS Rush是一个简单易用的工具，允许通过网络在多台机器上运行R3DS Wrap4D项目计算。R3DS Rush会在多个R3DS Node实例上自动分配和管理工作负载。
